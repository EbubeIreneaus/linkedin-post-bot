# Privacy Policy — LinkedIn Post Automation App This application is a personal automation tool used to publish content to LinkedIn on behalf of the authenticated user. --- ## Data Collected The application accesses: - Basic LinkedIn profile information - Permission to create posts on the user’s behalf No additional personal data is collected, stored, or shared. --- ## How Data Is Used Data is used solely to: - Authenticate the user with LinkedIn - Publish posts authorized by the user - Maintain API access tokens securely --- ## Data Storage Access tokens are stored securely and are **never shared with third parties**. --- ## Third Parties This application **does not sell, trade, or share user data**. --- ## User Control Users can revoke access at any time through their LinkedIn account settings. --- ## Contact For any questions about this policy: **Email:** help@zephyops.com
