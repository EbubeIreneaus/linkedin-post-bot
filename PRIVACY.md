Privacy Policy for LinkedIn Post Automation App

This application is a personal automation tool used to publish content to LinkedIn on behalf of the authenticated user.

Data Collected
The application accesses basic LinkedIn profile information and permission to create posts. No additional personal data is collected, stored, or shared.

How Data Is Used
Data is used solely to:
• Authenticate the user with LinkedIn
• Publish posts authorized by the user
• Maintain API access tokens securely

Data Storage
Access tokens are stored securely and are never shared with third parties.

Third Parties
This application does not sell, trade, or share user data.

User Control
Users can revoke access at any time via LinkedIn account settings.
